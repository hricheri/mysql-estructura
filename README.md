MySQL Estructura
----------------
Este repositorio contiene el modelado de dos diagramas de bases de datos del Nivel 1 de la Tarea S201 del Sprint 2 de Full Stack.

Archivos
________

1. Óptica (Cul d'Ampolla)
Modelo: Modelo Relacional Cul d'Ampolla.png
Script: Cul d'Ampolla.sql

2. Pizzería
Modelo: Modelo Relacional Pizzeria.png
Script: Pizzeria.sql

Tecnologías
-----------
MySQL Workbench para el modelado EER y generación de scripts.

Instalación
-----------
Descarga los archivos .png y .sql correspondientes.
